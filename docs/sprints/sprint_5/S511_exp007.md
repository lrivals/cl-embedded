## Objectif

Exécuter `scripts/train_unsupervised.py --model mahalanobis` sur les deux datasets en scénario domain-incremental et collecter les métriques CL complètes. C'est la première expérience à évaluer M6 Mahalanobis, et la première à tester un modèle non supervisé sur **Dataset 1 (Pump)** en plus du Dataset 2.

**Scénarios** :
- **Dataset 2** (Equipment Monitoring) : domain-incremental Pump → Turbine → Compressor (3 tâches, 4 features, cohérent avec exp_005)
- **Dataset 1** (Pump Maintenance) : domain-incremental par fenêtres temporelles (à définir avec S3-02 — ex. 3 périodes chronologiques)

**Critère de succès** :
- `experiments/exp_007_mahalanobis/results/metrics_mahalanobis_dataset2.json` et `metrics_mahalanobis_dataset1.json` présents avec `aa`, `avg_forgetting`, `bwt`, `auroc_final`, `ram_peak_bytes`, `inference_latency_ms`, `n_params`
- `ram_peak_bytes < 1024` sur les deux datasets (attendu ~80–160 B selon d)
- AUROC > 0.70 sur Dataset 2 (baseline minimum — exp_005 a montré > 0.90 pour K-Means et KNN)

### 1. Vérifier la sortie après exécution

```python
# Vérification post-run (à exécuter manuellement)
import json
import numpy as np
from pathlib import Path

exp_dir = Path("experiments/exp_007_mahalanobis/results")

REQUIRED_KEYS = ["acc_final", "avg_forgetting", "backward_transfer",
                 "auroc_final", "ram_peak_bytes", "inference_latency_ms", "n_params"]

for dataset in ["dataset2", "dataset1"]:
    fname = exp_dir / f"metrics_mahalanobis_{dataset}.json"
    with open(fname) as f:
        m = json.load(f)
    missing = [k for k in REQUIRED_KEYS if k not in m]
    assert not missing, f"{dataset} manque : {missing}"
    assert m["ram_peak_bytes"] < 1024, f"{dataset} RAM > 1 Ko : {m['ram_peak_bytes']} B !"
    print(f"{dataset} — OK | AA={m['acc_final']:.4f} | AUROC={m['auroc_final']:.4f} "
          f"| RAM={m['ram_peak_bytes']} B")

# Vérifier shape acc matrices
for dataset in ["dataset2", "dataset1"]:
    mat = np.load(exp_dir / f"acc_matrix_mahalanobis_{dataset}.npy")
    assert mat.shape == (3, 3), f"Shape inattendue : {mat.shape}"
    assert np.isnan(mat[0, 1]), "Case future non NaN"

print("Vérification exp_007 : OK")
```

## Diagnostics fréquents

| Symptôme | Cause probable | Solution |
|----------|----------------|----------|
| `LinAlgError: Singular matrix` | Features parfaitement corrélées | Augmenter `reg_covar` (1e-4 ou 1e-3) |
| `acc_final < 0.50` | Seuil inadapté (Task 0 atypique) | Ajuster `anomaly_percentile` ou vérifier normalisation |
| `auroc_final ≈ 0.50` | Distribution très chevauchante | Documenter comme limitation Gap 1 — Dataset non discriminant |
| `ram_peak_bytes >> 160` | Overhead tracemalloc (Python objects) | Comparer avec estimation analytique — acceptable si écart < ×100 |
| `KeyError: mahalanobis` dans `train_unsupervised.py` | Dispatcher non mis à jour | Appliquer S5-10, sous-tâche 5 |
| Dataset 1 non disponible | `data/raw/pump_maintenance/` absent | Vérifier téléchargement Kaggle + `.gitignore` |
